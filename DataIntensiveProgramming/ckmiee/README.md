# COMP.CS.320 — Personal repository

This is a personal repository created for all students in the Tampere University course COMP.CS.320 Data-Intensive Programming, Fall 2025.

The intention is for this repository to be used for storing your solutions to the weekly exercises. While the use of this repository is not compulsory, it is advisable to always use some kind of version control system to store your code.

Instructions on how to integrate this repository into the Databricks environment can be found from the course Moodle page: [Using Git repositories with Databricks](https://moodle.tuni.fi/mod/page/view.php?id=3503807).
